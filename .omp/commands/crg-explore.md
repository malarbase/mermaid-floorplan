Map an unfamiliar repo without burning tokens.

1. Call `list_graph_stats` and `get_architecture_overview` first.
2. Use `list_communities` and `get_community` for major modules.
3. Use `semantic_search_nodes` and `query_graph` for targeted exploration.
4. Always start with `get_minimal_context(task="<your task>")` and use `detail_level="minimal"` first.
